# Security Policy

## Supported Versions
Active development is on `main`. Security fixes may be backported to the latest release when applicable.

## Reporting a Vulnerability
Please report privately:
- Email: tech@allanswebwork.info
- Or open a **private** security advisory in GitHub (Security â†’ Advisories).
Do **not** file public issues/PRs with exploit details.

Include: affected commit/version, repro steps, impact, logs/PoC (if safe), suggested remediation.

## Response Targets
- Acknowledgement: **3 business days**
- Triage: **7 business days**
- Fix ETA: communicated post-triage; coordinated disclosure

## Safe Harbor
Good-faith, non-exploitive research on this repository is welcome. Avoid data exfiltration or service disruption.

## Disclosure
After remediation, we will publish a GitHub Security Advisory and credit reporters (unless anonymity requested).
